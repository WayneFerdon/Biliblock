# Biliblock

## Biliblock

Bilibili 网页首页卡片屏蔽等小功能

1. 卡片右下角添加屏蔽按钮，点击后将本地屏蔽该UP发布的视频
2. 统计并根据推荐次数显示背景色
3. 键盘操作视频卡片：

> `Shift`/`Numpad +`: `删除键`
> 
> `Ctrl`/`Numpad .`: `打开键`
> 
> `Alt`/`Numpad Enter`: `功能键`
>
> `空格键`/`Numpad 0`: `刷新键` - 全部不感兴趣并刷新
> 
> `刷新键`+`功能键`: 全部不感兴趣并删除
>
> `选项键`：点击不感兴趣（若有`撤销内容不感兴趣`键，则改为撤销）
> - `Numpad 1~9`/`QWEASDZXC`: 按照键盘位置将对应卡片
> - `1234567890-=`: 按照页面顺序将对应卡片
>
> `选项键`+`删除键`：在页面中删除视频卡片
>
> `选项键`+`功能键`：点不观看该作者，并添加到脚本blocklist（以后自动屏蔽和点作者和视频的不感兴趣）（若有`撤销作者不感兴趣`键，则改为撤销）
> 
> `选项键`+`打开键`：稍后再看
> 
> `选项键`+`打开键`+`功能键`：新标签页打开视频
